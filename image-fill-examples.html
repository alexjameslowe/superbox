Test 5
al-img-fill, greedy element, vertical conservation, default secondary dimension
There's a quirk with al-img-fill in in vertical al-greedy elements, which is that we can use the al-img-fill as a wrapper element so that the image won't start generating extra height. Or, you can place the al-img-fill on the al element without the wrapper, and the image will generate height. This is just one of the differences between vertical-space and horizontal space that we humans have. We don't mind scrolling vertically, but scrolling horizontally is generally a bad idea.
