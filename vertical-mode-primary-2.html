These examples show how vertical-mode elements deal with height. The rules for greediness are basically the same as the ones for horizontal-mode elements. The only differerce is that we have the sb-content-strict-height class.
Test 1
Basic example showing vertical-mode. We have two greedy elements, and the rest are frugal. Frugal elements with explicit heights get the al-explict class, just like in horizontal-mode

We have a horizontal-mode container on the left and inside that container, boxes switch from horizontal to vertical conservation, and the same rules of greediness apply. Note that just like the horizontal-mode examples, al elements that have an explicit height get the sb-explicit class. Note that the gold greedy box with the long content gets a scrolling bar, and that's because we're using sb-content-strict-height on that element.
